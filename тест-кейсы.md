# Тест-кейсы по КРО для игры "Морской бой 2.0"

## Общие данные

- URL: `http://isador.ru:8080/`
- Браузер: `Google Chrome`
- Учетная запись игрока 1: `LuckyAlbert / 123`
- Учетная запись игрока 2: `LuckyAlbert2 / 123`
- Роль зрителя: без авторизации

## Соответствие критериям

- Актуальность: каждый ТК привязан к конкретной КРО в поле `КРО` блока `Details`.
- Содержание: для каждого ТК заполнены `Precondition`, атомарные `Step`, конкретные `Test Data`, лаконичные `Expected Result`.
- Без ветвлений: внутри одного ТК описан один линейный сценарий проверки.
